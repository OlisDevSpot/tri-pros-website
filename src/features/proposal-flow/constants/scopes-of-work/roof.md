### Roof replacement - remove original clay tile roof, replace waterproofing, reinstall tile

~32 bsq.

Prepare house for new roof tear off: Cover ground around house, HVAC condensers as needed, fragile items and exposed ceilings in garage with nylon and tarps in order to collect debris from roof in an organized fashion.

- Provide dumpster to jobsite
- Remove existing tiles from roof and store away (tiles to be reinstalled)
- Demo existing roof underlayment to decking
- Clean wood deck from all nails and debris and prepare for new roof installation.
- Inspect the wood decking for structural soundness and replace wood to match as necessary.
- Replace up to 25% of damaged plywood
  - Replace rotten / damaged decking with new 1/2 inch plywood/osb boards as needed
- Clean all debris & haul away
- Pass tear off inspection with local city inspector

Install new roofing material per code: Prep work area, HVAC condensers as needed, cover with nylon and tarps in order to collect debris from roof in an organized fashion

- Install new synthetic underlayment - UDL30 or equivalent and fasten down with 1" plastic cap nails per code
- Install new powder coated 2x2/ 2x3 drip edge metal flashings along perimeter (Eaves/gutters and Rakes)- Color TBD
- Install starter tile around the perimeter per manufacturer standards
- Install ice & water shield (weather-lock) on all penetrations and valleys as needed.
- Install, seal and paint all new metal flashings, pipes vents to match
- Install new low profile attic vents to match
- Install new Owens Corning Cool Shingles- 25 yr Warranty Color to be selected by customer.
- Install decorative hip / ridge cap
- Clean up and haul away all debris
- Remove dumpster from job site

* 1 bsq.= 100 square feet
  *Any dry-rot repair is additional charge - $85/man hour + material
  *Permit fees will be billed separately. Contractor to pull and company to reimburse.
